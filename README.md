## 1. Tạo tiêu đề
> # Tiêu đề 1 (h1)  
> ## Tiêu đề 2 (h2)  
> ### Tiêu đề 3 (h3)  
> #### Tiêu đề 4 (h4)  
> ##### Tiêu đề 5 (h5)  
> ###### Tiêu đề 6 (h6)

## 2. Định dạng chữ

* **In đậm**
* ~~Gạch ngang~~
* `inlide code`
* Trích dẫn block code:
```php

```

## 6. Tạo hình ảnh
Cú pháp: `![mô tả](link)`

Có thể chụp ảnh màn hình và up hình đó lên trang http://i.imgur.com/ để lấy đường dẫn ảnh đưa vào Github
